# Krafto

> Visual editor for your existing React codebase.

Click any element in your live app to edit it, or ask the AI agent. Changes land in your source files as real diffs.

**Status: early preview.** `krafto init` connects your project, `krafto dev` runs the agent that streams your local dev server into the editor at [app.krafto.ai](https://app.krafto.ai). The visual editing layer is in active development.

## Try it

In a Next.js or Vite+React project (Node 18+, clean git tree):

```bash
npx krafto init   # detect the project and connect it to your krafto account
npx krafto dev    # start the agent — open the editor from your dashboard
```
